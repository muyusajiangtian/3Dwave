# 测试指南 / Test Guide

本文档描述如何测试 Magic Gesture 3D Controller 的多模态交互功能。

This document describes how to test the multimodal interaction features of Magic Gesture 3D Controller.

## 环境准备 / Prerequisites

### 必需条件 / Required

1. **浏览器**: Chrome 90+ 或 Edge 90+（Web Speech API 需要 Chromium 内核）
2. **摄像头**: 已连接并可正常工作
3. **麦克风**: 已连接（语音功能需要）
4. **权限**: 浏览器已授予摄像头和麦克风访问权限
5. **网络**: 语音识别可能需要网络连接（取决于浏览器离线模型支持）

### 启动服务 / Start Server

```bash
python server.py
```

在浏览器打开显示的URL（如 `http://localhost:8000`）。

## 测试2: 连续手势序列识别 / Gesture Sequence Recognition

**目标**: 验证DTW手势序列匹配正确触发复合命令。

### 准备
1. 点击右下角 🔧 按钮打开调试面板
2. 观察"手势序列流"区域实时显示手势段

### 测试用例

| # | 手势序列 | 预期命令 | 验证方法 |
|---|---------|---------|---------|
| 2.1 | 握拳(约0.5s) → 挥手(约0.3s) | "切换材质" | 模型表面质感改变（标准/金属/玻璃/陶瓷/霓虹循环） |
| 2.2 | 捏合(约0.4s) → 握拳(约0.4s) | "重置视图" | 模型回到初始位置和大小 |
| 2.3 | 快速挥手3次(中间有短暂停顿) | "显示帮助" | 调试面板显示/隐藏 |
| 2.4 | 握拳 → 捏合 → 挥手(各约0.3s) | "粒子爆发" | 粒子从模型中心发射 |
| 2.5 | 伸食指指向 → 握拳 | "切换线框" | 线框显示/隐藏切换 |

### 阈值调整测试

| # | 操作 | 预期结果 |
|---|------|---------|
| 2.6 | 将DTW阈值滑块调到0.3 | 匹配变得更严格，需要更精确的动作 |
| 2.7 | 将DTW阈值滑块调到0.8 | 匹配变得宽松，更容易触发但可能有误触 |

### 自定义模板测试

| # | 操作 | 预期结果 |
|---|------|---------|
| 2.8 | 点击"录制手势"→ 做一系列动作 → 点击"停止录制" | 弹出名称输入框 |
| 2.9 | 输入名称和动作 → 确认 | 调试面板显示"自定义模板: 1/5" |
| 2.10 | 重复录制的手势序列 | 触发对应动作 |
| 2.11 | 录制5个后再尝试录制 | 提示已达上限 |
| 2.12 | 点击"清空自定义" | 所有自定义模板被删除 |

## 测试4: 多模态融合 / Multimodal Fusion

**目标**: 验证手势和语音同时使用时的融合决策正确性。

### 同意图融合测试

| # | 操作 | 预期结果 |
|---|------|---------|
| 4.1 | 同时做"握拳→挥手"序列 + 说"切换材质" | 融合日志显示"FUSED"，材质切换（置信度比单模态高） |
| 4.2 | 同时做"捏合→握拳"序列 + 说"重置" | 融合日志显示"FUSED"，视图重置 |

### 冲突解决测试

| # | 设置 | 操作 | 预期结果 |
|---|------|------|---------|
| 4.3 | 优先级=最近优先 | 先做手势序列，立刻说不同命令 | 后触发的命令胜出 |
| 4.4 | 优先级=手势优先 | 手势和语音同时触发不同意图 | 手势意图执行 |
| 4.5 | 优先级=语音优先 | 手势和语音同时触发不同意图 | 语音意图执行 |

### 独立触发测试

| # | 操作 | 预期结果 |
|---|------|---------|
| 4.6 | 只用语音说命令（不做手势） | 语音独立触发动作 |
| 4.7 | 只做手势序列（不说话） | 手势独立触发动作 |
| 4.8 | 原有单手势控制（如握拳旋转） | 正常工作，不受融合影响 |

## 已知限制 / Known Limitations

1. Web Speech API 在某些环境下需要网络连接
2. 中文语音识别准确率取决于发音清晰度和环境噪音
3. DTW序列匹配对动作速度有一定容忍度，但过快或过慢可能不匹配
4. Firefox 不支持 Web Speech API，语音功能自动禁用
5. 自定义模板存储在 localStorage，清除浏览器数据会丢失
